randpint - GET RANDOM POSITIVE INTEGER

SYNOPSIS

    long randpint(long upper);

DESCRIPTION

Returns a positive integer with a user-specified upper limit.

Before calling randpint(), ensure the random number generator has been initialized by calling rinit().

PARAMETERS

INPUT

upper

The random number will be in the range [0..upper] inclusive.

RETURN

A random integer in the interval [0..upper].
